# ⚡ LumosPlay - Streaming Experience

O **LumosPlay** é uma plataforma de streaming temática inspirada no universo de Harry Potter. Este projeto foi desenvolvido
como um desafio prático durante a **Imersão Front-End da Alura**, onde o objetivo era criar uma interface funcional e
visualmente atraente baseada na experiência de usuário da Netflix.

---

## 🚀 Sobre o Projeto

O foco principal foi a criação de uma Single Page Application (SPA) que utiliza JavaScript puro para renderizar
conteúdos dinamicamente, permitindo que o usuário navegue entre categorias de filmes e séries com uma identidade
visual personalizada do Mundo Bruxo.

### 🧙‍♂️ Diferenciais e Personalização
Diferente de um projeto padrão, o LumosPlay traz:
- **Interface Temática:** Uso de vídeos de fundo (Quidditch) e elementos visuais que remetem à saga.
- **Seleção de Perfil:** Tela inicial personalizada para escolha de perfis baseados em gêneros cinematográficos.
- **Lumos/Nox (Dark Mode):** Implementação de um interruptor de tema para alternar entre os modos claro e escuro.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica focada em acessibilidade (uso de tags ARIA).
- **CSS3:** Estilização moderna com Flexbox, Grid e animações de transição.
- **JavaScript (ES6+):** - Manipulação de DOM para criação dinâmica de cards.
  - Uso de **ES Modules** (`import`/`export`) para organização do código.
  - Lógica de filtros para categorias e destaques.

---

## ♿ Acessibilidade (UX)

O projeto foi construído pensando na inclusão digital, utilizando:
- Atributos `aria-label` para navegação facilitada.
- Descrições em imagens e ícones.
- Semântica correta para leitores de tela.

---

## 📂 Estrutura de Arquivos

- `/assets`: Contém as imagens de componentes e perfis.
- `/js`: Módulos JavaScript (`data.js`, `Carousel.js`, `main.js`).
- `/catalogo`: Estrutura e estilos da página principal de conteúdo.
- `index.html`: Porta de entrada e tela de seleção de perfis.

---

## 🏁 Como visualizar

1. Você pode acessar o projeto rodando diretamente pelo link do **GitHub Pages** disponível na descrição deste repositório.
2. Para rodar localmente, basta clonar o repositório e abrir o arquivo `index.html` em seu navegador.
